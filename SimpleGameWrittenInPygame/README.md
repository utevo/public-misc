- `python3`
- `pygame`

```sh

$ chmod +x launcher
```

# Usage

```sh
$ ./laucher
```

# Controls

**ArrowLeft** - fly to the left

**ArrowRight** - fly to the right

**SPACEBAR** - shoot

**q** - turn off the game
